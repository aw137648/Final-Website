What This Class Is All About

This class teaches you all about HTML and CSS and a lot more. You get to make some websites in your own creative way, there's a lot to this class that is fun. This class also gives you a credit towards college, which is pretty cool. This will expand your knoweledge towards computers by a lot, and that could benefit you in the future, especially if you want a job that uses coding, for example: game developer, data engineer, ect. all of those use coding.

You'll be coding all sorts of things, and eventually get to build a website from scratch. It may sound hard but, it isn't, it's actually not that hard at all. Theres so much you'll get to explore and experience with this class. Overall this class is pretty cool, especially since you'll get a credit towards college.
